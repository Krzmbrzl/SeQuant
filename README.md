SeQuant: second quantization toolkit
=======

prerequisites:
  * CMake 3.9
  * a C++17 compiler
  * a recent Boost library
  * Range-V3 library (see https://github.com/ericniebler/range-v3.git)

for the impatient:
  * `cmake . -DRANGEV3_DIR=...`
    * `RANGEV3_DIR` = path to the top of Range-V3 source tree
  * `make test`
